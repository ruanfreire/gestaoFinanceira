# Exemplos TailAdmin (não usar em produção)

Componentes de demonstração do template TailAdmin. O app **Gestão Financeira** (`frontend/`) não importa estes módulos.

| Pasta | Conteúdo |
|-------|----------|
| `ecommerce/` | Widgets de dashboard demo |
| `form-elements/` | Showcase de inputs |
| `UserProfile/` | Cards de perfil demo |

Para novos componentes reutilizáveis, use `UI/src/components/ui/` e `UI/src/components/form/`.
